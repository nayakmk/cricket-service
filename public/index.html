API Overview

Explore cricket data through our comprehensive API

🏏 Matches (V2)

GET /api/v2/matches - Get all matches with pagination
GET /api/v2/matches/{displayId} - Get specific match details

👥 Teams (V2)

GET /api/v2/teams - Get all teams
GET /api/v2/teams/{id} - Get specific team information
POST /api/v2/teams - Create new team
PUT /api/v2/teams/{id} - Update team
DELETE /api/v2/teams/{id} - Delete team

🏃 Players (V2)

GET /api/v2/players - Get all players
GET /api/v2/players/{id} - Get detailed player statistics
POST /api/v2/players - Create new player
PUT /api/v2/players/{id} - Update player
DELETE /api/v2/players/{id} - Delete player
POST /api/players/merge - Merge duplicate players

📋 Team Lineups

GET /api/teamLineups - Get all team lineups
GET /api/teamLineups/{id} - Get specific lineup details

🎯 Live Scoring

POST /api/scoring/start-inning - Start a new inning
POST /api/scoring/record-ball - Record ball-by-ball data
